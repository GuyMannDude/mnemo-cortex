## What is Mnemo Cortex?
Originally built as "AgentB," Mnemo Cortex acts as an isolated, standalone memory sidecar for AI agents (like OpenClaw). It provides an ultra-fast L1/L2 vector cache hierarchy, resilient provider fallback chains (circuit-breakers), and true multi-tenant memory silo isolation.
