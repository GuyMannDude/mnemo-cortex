# Passport Eval Corpus

200 labeled examples across 4 files, generated by AL via ChatGPT Codex task
(PR #3 against sparks-router-v2, 2026-04-19). Moved here because Passport Lane
lives in mnemo-cortex, not sparks-router-v2.

## Files
- `benign.yaml` — 50 examples, all label=allow
- `toxic.yaml` — 50 examples, all label=hard_block
- `edge.yaml` — 50 examples, mix of review_required/local_only
- `adversarial.yaml` — 50 examples, mix of hard_block/review_required/local_only

## Schema
AL used the v0 product-spec schema: `observation.pattern`, `observation.context`,
`evidence[].quote`. Phase 1 code uses `observation.proposed_claim`,
`evidence[].excerpt`. See `tests/passport/corpus_migrate.py` for the adapter
that converts AL's shape into Phase 1 Observation objects.
